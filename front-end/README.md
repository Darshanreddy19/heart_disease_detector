# Heart Disease Detector - Frontend

Advanced heart disease prediction system with React and TypeScript.

## Project Setup

### Install dependencies
```bash
npm install
```

### Development
```bash
npm run dev
```

The frontend will be available at `http://localhost:8080`

### Build
```bash
npm run build
```

## Architecture

- **Framework**: React 18 + TypeScript
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **UI Components**: shadcn/ui
- **Build Tool**: Vite
- **Backend API**: http://127.0.0.1:5000

## Features

- Patient health assessment form with 13 medical parameters
- Real-time form validation with progress tracking
- Animated network background visualization
- ML model predictions with confidence scores
- Professional dark theme UI
- Smooth animations and transitions
